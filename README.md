# Netflix Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on an already existing Netflix dataset to identify patterns and trends in Netflix content. The analysis focuses on understanding content distribution based on type, genre, release year, and country.

## Dataset
- Source: Kaggle – Netflix Movies and TV Shows
- Format: CSV
- Number of records: 3000
- Features:
  - Title
  - Type (Movie / TV Show)
  - Genre
  - Release Year
  - Rating
  - Duration
  - Country

## Tools and Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Visual Studio Code

## Analysis Performed
- Distribution of content types (TV Shows vs Movies)
- Release year trend analysis
- Genre-wise content distribution
- Country-wise content contribution

## Key Findings
- TV shows are more prevalent than movies in the dataset.
- A significant portion of content has been released in recent years.
- Horror and Thriller are among the most common genres.
- Netflix content is produced across multiple countries, indicating a global content strategy.

## Conclusion
The exploratory data analysis provides insights into Netflix’s content library and highlights trends in content type, genre preferences, release patterns, and country-wise distribution. This project demonstrates practical use of Python for data analysis and visualization.
